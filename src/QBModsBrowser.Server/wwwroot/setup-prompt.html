Welcome to QBMBAMM

Enter the full path to your Starsector mods folder so the browser can read your installed mods.

Keyboard shortcuts

When a text field is not focused.

A D
Mod list: previous / next page. Mod page: previous / next mod.
Q E
Mod list: previous / next category filter. T Mod list: open this popup. S Mod page: back to the list.

What else is in the app

Browse, filter, sort Search, minimum game version, category, install status, age window, sorting. Automatic download/install Automatically install most mods straight from the forum. Mod managing Enable, disable, or remove mods in your folder. Automatic extract/install In installs section, press "Extract" to extract and install archives in your mods folder. Mod profiles Header bar profiles to enable/disable groups of local mods. Update checking Checks for updates through the forum, ModRepo, and VersionChecker data. Favorites Star mods on cards. Dependencies Displays missing dependencies. Control Panel Scrape for new forum data, adjust settings.

This can be changed later in the Control Panel.
